Requirements for using Flash movies in Authorware

To author a piece that uses the Flash Asset Xtra, you need:

Authorware 5 Attain
Windows 95 or NT 4.0

To playback a piece that uses the Flash Asset Xtra, you need:

Authorware 5 Attain
On Windows: Windows 98, Windows 95, or Windows NT 4.0
On the Macintosh: a computer with a PowerPC processor

You cannot use the Flash Asset Xtra with Windows 3.1 or with non-PowerPC Macintoshes.

All Flash movies must be:

Created with Macromedia Flash version 2.0 or later
Saved as a Flash Player movie
