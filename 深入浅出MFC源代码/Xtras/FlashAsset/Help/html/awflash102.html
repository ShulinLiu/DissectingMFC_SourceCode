Overview

The Macromedia Flash Asset Xtra opens up the world of vector-based animation to Macromedia Authorware and web-packaged Authorware pieces. After creating a movie in Macromedia Flash, just import it into Authorware and use it as you would any other type of media. Effects that once required multiple versions of a bitmapped image—such as rotating an image—can now be accomplished with a single, small Flash movie.

This guide provides everything you need to get started.
