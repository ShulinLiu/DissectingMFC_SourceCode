What you can do with the QuickTime Xtra

Using the QuickTime Xtra, you can add QuickTime media to your Authorware pieces. Here are some of the most important features:

The QuickTime Xtra provides seamless integration of QuickTime media. Existing QuickTime movies will continue to work as they currently do.
You can play QuickTime media in any Authorware piece. You can even deliver your piece over the World Wide Web using the Authorware Web Player.
QuickTime plays a wide array of digital video, animation, sound, and bitmap formats, which you can now incorporate in your Authorware pieces using QuickTime sprite icons. For a list of formats supported by QuickTime, see the Apple QuickTime web site.
You can set starting and ending points within QuickTime media and loop playback between those points.
